# AlignAssist Project

## Purpose
A static website for "Align Assist" — a virtual assistant / medical billing & coding services company.

## Tech Stack
- Plain HTML, CSS, JavaScript (no framework)
- Pages: index.html, services.html, blog.html
- CSS files: styles.css (main), services-page.css, blog.css
- JS files: script.js, services-page.js, blog.js
- Assets: logo.png, video files in root, Images/ folder

## Structure
- Single directory, no build system
- Footer background: #000000
- Uses CSS custom properties (--primary-color, --secondary-color, --text-secondary)
- Responsive with media queries

## Commands
- System: Windows (use `dir`, `type`, etc.)
- No build/lint/test commands — static site, just edit and refresh
